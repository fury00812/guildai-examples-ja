# guildai-examples-ja
実験管理ツールGuild AIのコードサンプル

## 要件
- Python 3.7
- Pytorch 1.4

## pipenvによる環境構築
リポジトリをクローンし, パッケージをインストール
```
git clone https://github.com/fury00812/guildai-examples-ja.git
cd guildai-examples-ja
pipenv install
``` 
仮想環境をアクティベート
```
pipenv shell
```

- pipenvを使用しない場合は, requirements.txtに記載の必要パッケージをインストールする.
